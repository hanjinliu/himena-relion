# himena-relion

`himena-relion` is a modern RELION GUI implemented as a
[himena](https://github.com/hanjinliu/himena) plugin.

For the details of the jobs and algorithms, please refer to the
[RELION documentation](https://relion.readthedocs.io/en/latest/).
